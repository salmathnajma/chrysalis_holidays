=== CA Bridge ===
Description: CA Bridge helps to connect different postypes. An extension for cappa theme.
Version: 1.0
Author: Nithin K V
Author URI: http://creativeasset.net/
Requires at least: 3.0
Tested up to: 4.9

-> Change Parent,Child post types and Category in config.php. Take a backup of this file before updating the plugin
-> Change Image sizes in plugin settings. (Settings -> Bridge Settings)
-> Change Archive column width in plugin settings. (Settings -> Bridge Settings)
-> If you want to change the html of the template, please copy the template file from templates folder and create  a 'ca-bridge' folder in your theme and paste it.
